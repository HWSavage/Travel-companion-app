    One-time setup
    1
    Go to console.cloud.google.com → New Project → name it Holiday App

    APIs & Services → Library → enable Google Sheets API and Google Drive API

    Credentials → Create OAuth 2.0 Client ID → Web application → add your app URL to Authorised JavaScript origins
    4
    Copy the Client ID, paste above → Save → Sign in with Google
